### 回流reflow 重绘repaint

重绘是当节点需要更改外观而不会影响布局的，比如color就叫成为重绘  背景色

回流是布局或者几何属性需要改变就称为回流（需要计算他们在设备视口内的确切位置和大小 加一个节点的高度

回流会发生重绘、重绘不一定会引发回流。回流所需的成本比重绘高的多，改变深层次的节点很有可能导致父节点的一系列回流

所以一下几个动作可能会导致性能问题

1. 添加或删除可见的dom元素

2. 元素的位置发生改变

3. 元素的尺寸发生变化（外边距，内边框，边框大小，高度和宽度

4. 内容发生变化，比如文本变化或图片被另一个不同尺寸的图片所替代。

5. 定位或者浮动

6. 浏览器的窗口尺寸变化（回流是根据视口的大小来计算元素的位置和大小的

减少回流和重绘

1. 批量修改dom（虚拟dom减少了dom操作 

2. 对于复杂动画效果，使用绝对定位让其脱离文档流

3. css硬件加速 GPU teansfom，opacity，filters这些动画不会引起回流重绘。
