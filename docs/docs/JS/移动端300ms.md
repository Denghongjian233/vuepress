产生原因

用户点击之后需要判断用户是否需要双击 手机会等待300ms

解决：插件 fastClick.js

FastClick的实现原理是在检测到touchend事件的时候，会通过DOM自定义事件立即出发模拟一个click事件，并把浏览器在300ms之后的click事件阻止掉。
