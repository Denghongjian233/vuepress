箭头函数没有自己的prototype（原型），也没有自己的this指向，更不可以使用arguments参数，不能使用new、它会捕获其所在（即定义的位置）上下文的this值， 作为自己的this值

new 操作符的原理

1. 创建一个对象

2. 将构造函数的作用域赋值给新对象 新对象的指针 __proto__ 指向 构造函数prototype

3. 为这个新对象添加属性和方法

4. 返回新对象。

谁调用指向谁
